При запуске программы отображается меню пользователя в котором можно выбрать пользователя, сменить язык вывода результата или добавить нового пользователя
Если программа запускается первый раз, выходит предупреждение что БД пустая и появляется меню по созданию первого пользователя, в нем необходимо указать имя, токен, секретный ключ и базовый адрес для доступа к сервису. Если в поле адреса не указывать значение по умолчанию будет установлен адрес "dadata.ru"
После выбора/смены пользователя или выбора языка вывода результатов появляется меню для поиска по адресу, в нем можно осуществить поиск по адресу или вернуться в меню пользователя.
При выборе пункта меню поиск по адресу, необходимо ввести адрес в произвольном формате и нажать Enter, сервис вернет результаты по указанному адресу, необходимо из предложенного списка выбрать номер того адреса который наиболее подходит к заданному поиску и нажать Enter. Сервис вернет координаты и полный адрес по выбранному значению. После этого программа вернется к меню поиска по адресу.
В каждом меню есть возможность завершить выполнение программы
